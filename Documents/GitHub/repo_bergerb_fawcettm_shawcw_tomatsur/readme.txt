This project can be ran by the use of GUI through qt.
  
This project was intended for the company called Circle of Bees in which there are three types of users; Researchers, Teachers,Hive Owner.  The user will navigate through the GUI, can either configure their hive, run a real time simulation of a hive, or graph their hive.  We implemented SQL database for the Hives.  Our project generates and decodes data based on the Bee Activity that each hive will have.  


1.  To run this project run the GUIpackage subproject and then a GUI will appear.  
2.  The user can choose which user type he/she is through the GUI.
3.  The user can run a simulation of the bee hive, configure their own bee hive, or Graph Multiple Bee Hive Activity through the GUI.
